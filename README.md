## Getting Started

**Node.js version is 16.14.2**

1. Clone the repo
   ```sh
   git clone https://github.com/Dermatit/contacts-app.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Start app and json-server
   ```sh
   npm run dev
   ```
