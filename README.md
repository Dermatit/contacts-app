## Getting Started

Node.js is v16.14.2

1. Clone the repo
   ```sh
   git clone https://github.com/Dermatit/contacts-app.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Start contacts-app and json-server
   ```sh
   npm run dev
   ```
